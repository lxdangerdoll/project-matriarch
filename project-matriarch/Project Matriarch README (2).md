# **PROJECT MATRIARCH (The Matilda Pilot)**

**Sovereign DBT Coach & Cognitive Shield**

*"The Ethical Compiler is the hospital. Project Matriarch is the field medic."* Project Matriarch (codenamed "Mariposa") is an open-source, client-side React application designed to function as an on-demand Cognitive Shield for survivors of interpersonal trauma, domestic interference, and high-control environments. It utilizes expert-prompted Large Language Models (LLMs) to facilitate real-time Dialectical Behavior Therapy (DBT) skills coaching during "8/10 distress" events.

## **🛡️ The AEGIS Ecosystem**

Project Matriarch is part of the broader **AEGIS** harm-reduction suite:

* [**The Ethical Compiler**](https://lxdangerdoll.github.io/ethical-compiler/) **(The Hospital):** A forensic auditing tool used to deconstruct manipulative rhetoric and theological abuse post-incident.  
* [**Project Matriarch**](https://lxdangerdoll.github.io/project-matriarch/) **(The Field Medic):** A tactical, real-time stabilization tool used to manage acute somatic recoil and prevent emotional spiraling *during* an incident.

## **🧠 Core Methodology: The Language Bypass**

During an acute trauma trigger, the amygdala floods the nervous system (the "Somatic Scream"). Matriarch forces a "Language Bypass." By requiring the user to articulate their distress and engage with structured DBT protocols (STOP, TIPP, DEAR MAN) via text, the system pulls neural focus back to the prefrontal cortex (the "Architect" brain), facilitating rapid de-escalation.

### **The Biesma Safeguards (Epistemic Security)**

Unlike standard commercial LLMs optimized for sycophancy (agreeableness), Matriarch is hard-coded with strict epistemic safeguards:

1. **Anti-Sycophancy Rigor:** It will not validate "philosophical spiraling" or narrative drift.  
2. **Somatic Priority:** If it detects severe distress, it overrides conversational logic and directs the user to immediate physical regulation (e.g., Temperature, Paced Breathing).  
3. **Sovereign Record:** It does not harvest data. All logs are local, allowing the user to download their chat history to share with human clinicians, circumventing institutional redaction.

## **✨ Features**

* **Bring Your Own Key (BYOK):** Users input their own Google Gemini API key via local storage, ensuring complete privacy and zero centralized database tracking.  
* **The Canon Vault:** Pin breakthrough insights and save them locally to build a personal "Sovereign Record" of survival skills.  
* **Vocalis Synthesis:** Integrated Text-to-Speech (TTS) using Gemini's audio models for auditory grounding.  
* **Context Injection:** A customizable "Flock's Plan" sidebar allows users to define their core triggers, diagnoses, and preferred DBT skills, which are silently injected into every API call.  
* **Distress Override:** A "Heartbeat" panic button that initiates a 60-second visual breathing overlay (The STOP Protocol).  
* **Audit Downloads:** Export single insights or full session transcripts as .txt files for clinical hand-offs.

## **🛠️ Tech Stack**

* **Framework:** React 18 \+ Vite  
* **Styling:** Tailwind CSS (configured for deep-space/cyberpunk aesthetics)  
* **Icons:** Lucide React  
* **AI Engine:** Google Gemini API (gemini-2.5-flash-preview-09-2025 with gemini-2.5-flash-preview-tts)  
* **Persistence:** HTML5 localStorage (No external databases)

## **🚀 Quick Start (Development)**

1. Clone the repository:  
   git clone \[https://github.com/lxdangerdoll/project-matriarch.git\](https://github.com/lxdangerdoll/project-matriarch.git)

2. Navigate to the directory:  
   cd project-matriarch

3. Install dependencies:  
   npm install

4. Start the development server:  
   npm run dev

5. Click the **Settings (Sliders)** icon in the header to input your Gemini API Key.
